A sleek and interactive stopwatch web application built with HTML, CSS, and JavaScript. This project features two versions of the stopwatch with different designs and functionalities.

## Features

### Version 2 (Modern Design)
- Clean, modern user interface with smooth animations
- Precise timing with centisecond accuracy
- Start, pause, and reset functionality
- Lap time tracking with split times
- Automatic highlighting of fastest and slowest laps
- Responsive design for all device sizes
- Custom scrollbar for lap times
- Visual feedback for button states
- Modern color scheme with dark mode

### Version 1 (Classic Design)
- Simple and straightforward interface
- Basic stopwatch functionality
- Lap time recording
- Basic styling

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (ES6+)
- Font Awesome Icons

## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/blackpython34/SCT_WD_2.git
```

2. Navigate to the project directory:
```bash
cd SCT_WD_2
```

3. Open either version in your browser:
- Modern version: `index2.html`
- Classic version: `index.html`

## Usage

### Modern Version (index2.html)

1. Click the play button (green) to start the stopwatch
2. Click the flag button (blue) to record lap times
3. Click the play/pause button to pause the stopwatch
4. Click the reset button (red) to reset everything
5. View lap times in the scrollable list below
   - Fastest lap highlighted in green
   - Slowest lap highlighted in red

### Classic Version (index.html)

1. Click "Start" to begin the stopwatch
2. Click "Lap" to record lap times
3. Click "Reset" to clear everything
